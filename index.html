What is Graphify?

Outdo yourself - create the most impressive Graph Gist ever

Join our GraphGist competition this month. We want to encourage you to model a domain that you know / like in a graph model and describe it interactively in a GraphGist. The rules

How does it work?

Find a great domain (Music, Dating, Comics, Healthcare, Politics, Sciences, ...)

Whiteboard a good example graph model.

Determine interesting use-cases.

Create Cypher statements for setup and query-use-cases.

The graphgist needs to be public, so the ASCIIDOC course can be shown.

Write a good description and create a pretty model picture.

Compose it all nicely in your asciidoc-graphgist-file.

How do I participate?

Tweet the public URL of your graphgist and add the tags #graphify and #graphgist to the tweet.

Add your gist under the section "Graphgist competition" in the graphgist collection Wiki (see "resources" in the graphgist header)

The GraphGist needs to be registered before Sept 30, 2013
